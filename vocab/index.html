Directory of EPUB Vocabularies, Registries and Reserved Prefixes

Vocabularies

EPUB Structural Semantics Vocabulary
Defines structural semantics for use in the epub:type attribute in EPUB Content Documents.
EPUB 3 Magazine Navigation Vocabulary
Defines semantics for use in region-based navigation.
Media Overlays Vocabulary
Defines properties related to Media Overlays for use in the Package Document.
EPUB Meta Properties Vocabulary
Defines properties for use with the Package Document meta element's property attribute.
EPUB Manifest Properties Vocabulary
Defines properties for use with the Package Document item element's property attribute.
EPUB Spine Properties Vocabulary
Defines properties for use with the Package Document itemref element's property attribute.

Registries

EPUB Link Relationships Registry
Defines values for the link element in the Package Document metadata.
EPUB Registry of Collection Roles
Defines values for collection elements in the Package Document.
EPUB Registry of Publication Types
Defines publication identifiers for the dc:type element in the Package Document metadata.

Reserved Prefixes

EPUB Publications Reserved Prefixes
List of prefixes that can be used in Package Document metadata without having to be declared in the prefix attribute.
EPUB Content Documents Reserved Prefixes
List of prefixes that can be used in the epub:type attribute in EPUB Content Documents.
